Script 1: This script demonstrates the use of the "new" operator to create an instance of a user-defined object type.

If you view the source for this page, you will find a function called ADT, which has properties and methods. ADT is being used as the user-defined object type. Clicking the "here" link below calls a function called setdata which creates a new instance of ADT and then sets the properties of the new ADT object via the prompt and confirm functions. The prompt function is a built-in javascript function which requests data from the user. The confirm function is a built-in javascript function that returns true or false. Once the data are entered the setdata function calls the showdata method of the new ADT object and the user-entered data are displayed in there own window.
