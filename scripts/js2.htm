Script 6: Time and date.


Note:  The lastModified property is derived from the HTTP header data sent by the web server. Servers generally obtain this date by examining the file’s modification date. The last modified date is not a required portion of the header, and some servers do not supply it. If the server does not return the last modified information, JavaScript receives a 0. The code above checks the date returned by lastModified and prints out a value that corresponds to unknown if it receives a ZERO.
