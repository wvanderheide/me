Note: The database selected must have read permissions on the msysobjects table, in order for this application to work.

To set the permissions on the msysobjects table do the following:

  1. First, open the Access DB on your computer.
  2. Next, make the system tables visible by selecting "options" from the tools menu and clicking the view tab and then checking show for system objects.
  3. Finally, set the permissions to read, by selecting "security " from the tools menu. And then choosing "User and Group Permissions". Find the object (table) of interest and set it's permissions. In this case the object of interest is the msysobjects table.
