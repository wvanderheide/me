The "Document Manager" is an document management and retrieval system written in ASP in 2000. I wrote all the ASP code, as well as all of the javascript verification code. This tool allows administrators to upload and categorize documents. Front-end users can then retrieve the documents by various search options. The pictures to the left show the search results page as the big picture, and the two small pictures are the add document screen, and the search module.

This tool is part of an Intranet site and so can not be viewed on-line.
