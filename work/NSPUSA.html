What I did...

This is the main ecommerce website for Natures Sunshine USA. It is written ASP.NET (C#), and consumes WCF services. For the site I developed the members portal, which uses AJAX. The content of this site is edited and upkept via the CMS system we developed at NSP, of which I developed the menu management section. Custom inherited User Controls were developed for this site as well, for example phone number and SSN text boxes.
