What I did...

This ASP.NET (C#) tool is part of an e-mail system I developed in 2006. The system consists of a webservice, that sends emails and logs them them in a SQL Server Database, and this administration tool. This administration tool allows administrators to get email system statistics, find sent emails, set up HTML templates for the emails, save and edit "prepared" email messages, add attachments to emails, and test the webservice.
