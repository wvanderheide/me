What I did...

Using ASP I developed an efficient way to manage documents on this site. Users have the ability to search for a document, view documents by category, or to view all documents. The web-master is able to update the site dynamically by simply filling out an on-line form.

This is an Intranet site and so can not be viewed on-line.
